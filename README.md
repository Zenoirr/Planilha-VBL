# 🏐 Planilha VBL

Sistema de análise de desempenho para jogadores de vôlei, focado em **scout simples, rápido e justo**.

🔗 Acesse: https://zenoir.github.io/planilhavbl/

---

## 📊 Sobre o projeto

A Planilha VBL foi criada para facilitar a avaliação de jogadores durante partidas ou treinos, trazendo:

- Controle de **pontos feitos (scored)**
- Pontos sofridos (**scored on**)
- Erros ofensivos (**mistake OF**)
- Erros defensivos (**mistake DEF**)
- Cálculo automático de **rating (0 a 10)**
- Sistema de **eficiência e consistência**
- Destaque de **MVP e Worst player**
- Suporte a **até 3 sets + resultado final**

---

## 🎯 Objetivo

Criar uma ferramenta justa onde:

- Fazer pontos é valorizado 📈  
- Erros impactam, mas não destroem o jogador ⚖️  
- Jogadores não são punidos por funções (ex: setter)  

---

## 🧠 Como funciona o rating

O rating vai de **0 a 10** e considera:

- Pontos feitos (principal peso)
- Pontos sofridos
- Erros (OF + DEF)
- Eficiência geral

👉 O sistema foi ajustado para:
- Evitar ratings muito baixos com facilidade
- Valorizar impacto positivo no jogo
- Ser mais motivador para os jogadores

---

## 🧩 Funções por posição

- **Ponteiro / Oposto / Líbero / DS Spiker**
  - Influenciam com pontos, erros e defesa

- **DS Tsk / Setter**
  - Não são obrigados a pontuar
  - Pontuar = bônus
  - Erros impactam levemente
